Functional groups are specific atoms or bonds that determine a molecule's chemical properties.

## 1. Quick Reference Table
| Class | Structure | Suffix | Example |
| :--- | :--- | :--- | :--- |
| **Alcohol** | $-OH$ | *-ol* | Ethanol |
| **Ether** | $-O-$ | *-oxy-* | Methoxyethane |
| **Aldehyde** | $-CHO$ | *-al* | Ethanal |
| **Ketone** | $C=O$ | *-one* | Propanone |
| **Acid** | $-COOH$ | *-oic acid* | Ethanoic acid |
| **Ester** | $-COOR$ | *-oate* | Ethyl ethanoate |
| **Amine** | $-NH_2$ | *-amine* | Methanamine |
| **Amide** | $-CONH_2$ | *-amide* | Ethanamide |

## 2. Priority Hierarchy (for Naming)
1. Carboxylic Acid (Highest)
2. Ester / Amide
3. Aldehyde / Ketone
4. Alcohol / Amine
5. Alkenes / Alkynes
6. Halogens / Alkyls (Lowest)
